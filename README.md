# wam-rt-robotics

Realtime robotics dataviz for the West African Margin Project

## Overview
This repository provides code for the realtime visualisation of data relevant for the Southen Benguela [West African Margin Project](https://westafricanmargin.com/).

## Scope
- Sat Maps
- Slocum Science Data Viz
- Waveglider Science Data Viz

*Scope is subject to change*
